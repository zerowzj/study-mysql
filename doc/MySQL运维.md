# 一. 安装

## 1.1 二进制安装

1. 解压

   ```shell
   tar -zxvf mysql-5.7.29-linux-glibc2.12-x86_64.tar.gz -C /usr/local/mysql
   ```

2. 初始化

   ```shell
   ./mysqld --initialize \
       --user=mysql \
   	--basedir=/usr/local/mysql \
   	--datadir=/usr/local/mysql/data
   ```

3. 撒旦法

   ```
   Qh=UXyJqP9_f
   ```
